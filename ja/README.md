- 24時間稼働していますが、05:00、17:00 に自動的に再起動がかかります。

- 統合版でアクセスしても、このサーバでは Java 版の仕様で動作します。
- 一部、Java 版でも通常とは異なる動作をする場合があります。
- 描画距離の上限は 16 チャンク、演算距離の上限は 8 チャンクです。
